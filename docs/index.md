# ACCESS Notification Catalog

Welcome to the ACCESS Notification Catalog — an auto-generated reference of all automated notifications sent from ConnectCI portals.

This catalog is automatically generated from the ConnectCI Drupal application and deployed to GitHub Pages. Each section documents the notifications sent by a specific portal or service area.
